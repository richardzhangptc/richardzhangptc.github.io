Description

"Untitled Ship Game" aka "Nucleocapsid" is an immersive co-op adventure where 3 players control a capsule ship sent to save a planet ravaged by a spacefaring virus. The virus is located at the planet's core and has weaponized the local biomass to create hostile alien creatures. The players embody three human brains suspended in a vat of liquid within the pod, tasked with infiltrating the planet and destroying the virus core. Each player controls a specific subsystem of the ship and cooperation is necessary for success. The enemies encountered are biological entities designed to mimic an immune system.

History

This project was developed by a group of students for a final project assignment in the class ECS179 at UC Davis. It showcases the accumulated game development skills of the team. This game was made with the Godot engine.

Features

  • Co-op gameplay with up to 3 players, each controlling a different subsystem of the ship.
  • Immersive storyline with unique enemy designs.
  • Dynamic environments that react to player actions.
  • Various upgrades to improve the players' ship.
  • Replayability with controlling different ship subsystems.
